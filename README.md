# NutriPlan

AI-powered meal planning and grocery management app. Enter your dietary goals once and get a personalized 7-day meal plan, a categorized grocery list, and product recommendations — all generated by Claude AI using your local supermarket's catalog.

![NutriPlan](public/leaf.svg)

## Features

- **Personalized onboarding** — set your calorie target, goal (gain / lose / maintain), dietary preferences (halal, vegan, vegetarian, gluten-free), allergies, supermarket, and weekly budget
- **7-day AI meal plan** — breakfast, lunch, dinner, and snack for every day, with macros, prep time, and 2 alternatives per meal
- **Smart grocery list** — auto-generated from your meal plan, grouped by category, with swap suggestions and price estimates
- **Meal swapping** — pick an alternative meal and the grocery list updates automatically
- **Real nutrition data** — product catalog enriched with per-100g calories, protein, carbs, and fat from USDA FoodData Central
- **Export** — download the meal plan as CSV or print the grocery list as a PDF
- **Persistent sessions** — your plan survives page refreshes; nothing is lost until you reset

## Tech stack

| Layer | Choice |
|---|---|
| UI | React 18 + Vite 6 |
| Styling | Tailwind CSS 3 + CSS custom properties |
| Fonts | Fraunces (display) + DM Sans (body) |
| AI | Claude Sonnet (`claude-sonnet-4-6`) via Anthropic API |
| Catalog | Lidl.fr scraper (Playwright) + USDA FoodData Central |

## Getting started

### 1. Clone and install

```bash
git clone https://github.com/imran-tag/nutriplan.git
cd nutriplan
npm install
```

### 2. Add your Anthropic API key

Either create a `.env.local` file:

```
VITE_ANTHROPIC_API_KEY=sk-ant-...
```

Or enter the key directly in the app's setup screen on first launch.

### 3. Run

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173), complete the 4-step onboarding, and your meal plan generates automatically.

## Scraper (optional)

The scraper fetches the Lidl.fr product catalog and enriches each item with nutrition data from USDA FoodData Central. The output (`scraper/lidl_catalog.csv`) is already committed so you don't need to run it unless you want fresh data.

```bash
cd scraper
npm install
npm run setup        # installs Playwright's Chromium browser
node scraper.js
```

The scraper requires a USDA FoodData Central API key (free at [fdc.nal.usda.gov](https://fdc.nal.usda.gov/api-guide.html)). Set it in `scraper/scraper.js` under `CONFIG.usdaApiKey`.

### CSV columns

| Column | Description |
|---|---|
| `name` | Product name |
| `brand` | Brand (if any) |
| `category` | Lidl product category |
| `price` | Regular price (€) |
| `lidlPlusPrice` | Lidl Plus member price |
| `packagingQty` | Pack size number |
| `packagingUnit` | Pack size unit (g, ml, pcs…) |
| `compareUnit` | Comparison unit (kg, L…) |
| `pricePerUnit` | Price per comparison unit |
| `cal` | Calories per 100g (USDA) |
| `protein` | Protein per 100g (USDA) |
| `carbs` | Carbohydrates per 100g (USDA) |
| `fat` | Fat per 100g (USDA) |
| `link` | Product page URL |
